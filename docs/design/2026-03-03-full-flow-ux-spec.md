**SMS 인증 마이크로 인터랙션:**
- 전화번호 입력 시: 숫자 입력마다 자동 하이픈 포맷팅 (`010-1234-5678`)
- "인증번호 받기" 탭: `HapticService.medium()` + 로딩 스피너
- 인증번호 발송 성공: 캐릭터 말풍선 텍스트 `AnimatedSwitcher`로 전환
- 코드 입력 박스: 각 숫자 입력 시 `HapticService.selection()` + 박스 보더 earthColor로 전환
- 6자리 완성: 자동 검증 시작 → 성공 시 체크마크 애니메이션 (Lottie or custom) + `HapticService.success()` → 0.3s 후 자동 진행
- 검증 실패: 박스 전체 shake 애니메이션 + fireColor 보더 + `HapticService.error()`
- 타이머: 3분 카운트다운, 0 도달 시 "인증번호 재발송" 활성화

## 6. 데이팅 프로필 셋업 (DatingProfileSetupPage) — **신규 설계**

### 핵심 UX 원칙

**"지루한 프로필 작성이 아닌, 나의 매력을 발견하는 재미있는 과정"**

- 한 화면에 1~2개 질문 (토스 스타일)
- 캐릭터가 질문하고 격려하는 구조
- 칩 선택형을 최대한 활용 (타이핑 최소화)
- 스킵 가능한 항목 명시
- 진행 바가 빠르게 차오르는 느낌 (8단계를 체감상 짧게)

### 경로 추가 필요
```dart
// app_constants.dart
static const datingProfileSetup = '/dating-profile-setup';
```
